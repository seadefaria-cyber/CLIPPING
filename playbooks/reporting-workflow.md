# Reporting Workflow

How weekly reports get generated and sent to clients.

## Tool: Metricool ($35/mo)

Connect all campaign accounts in one Metricool dashboard.

## What the Weekly Report Shows

1. **Total views** across all accounts (the big number clients love)
2. **Top 3 performing clips** with view counts
3. **Follower growth** per account
4. **Platform breakdown** (which platform drives the most reach)

## Process

1. Metricool auto-generates a branded PDF report weekly
2. Sean reviews it (1 min)
3. Sean forwards it to the client via email (1 min)
4. Done.

## If a Client Wants More Detail

- Metricool has CSV export for raw data
- Can build custom Google Sheets dashboards if needed
- But start simple. The PDF is enough for most clients.
